Perlin tublence toy uses Perlin noise to make interesting 2D art.

3 kinds of functions are used to make an image:

  • Base function: function for every point (x,y) generates a value between 0 and 1, such as a cosine ripple
  • Noise function: smooth perlin noise function for every point (x,y) generates a pseudo random number between 0 and 1
  • Color function: for a value between 0 and 1, defines a color

How the image is calculated: Two noise functions N1 and N2 are used. For every point (x,y) the value of the noise functions (0 to 1) are calculated. Some mulpiple of these noise functions is added to (x,y) to get a new point (xt,yt) with turbulence added. This new point (xt,yt) is then the input to the base function, which generates a number b between 0 and 1. The output of the base function is then the input to a color function, that determines the color of the point at (x,y).

Tips

  • The best way to see what a base function looks like is to click the "noise" tab and set the noise magnitude to it's lowest setting
  • High noise magnitude settings make crazy images!
